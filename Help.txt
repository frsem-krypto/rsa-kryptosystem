  1.Richtiges Verzeichnis Wählen:
    cd rsa-kryptosystem

   !! WINDOW-HINWEIS: NICHT den kompletten Pfad zur Datei, wie z.B. cd G:\Seminar\rsa-kryptosystem, sondern Schritt für Schritt, in diesem Fall:
               cd G:
               cd Seminar
               cd rsa-kryptosystem !!

  2.Projekt downloaden:
    git clone https://github.com/PixelProfi/rsa-kryptosystem.git

  3.Die zu bearbeitende Datei "fokusieren":
    git add >entsprechende Datei<

  4.Datei pullen (aus dem Internet aktualisieren):
    git pull

  5.Datei öffnen und bearbeiten.

  6.commiten:
    git commit -m "Info/Veränderung"

  !! Bei Fehlermeldung sicherheitshalber noch einmal pullen. !!

  7.Datei pushen (In die Cloud wieder hochladen):
    git push
    -> Benutzername
    -> Password



Alle Commands:
git clone https://github.com/PixelProfi/rsa-kryptosystem.git
ls = list
git status = aktuellen Status der Datei anzeigen
git pull
git push
strg + x = abbrechen
strg + s = speichern

PS: Bei dem Download von Git auf Windows bekommt man auch eine eigene Git-Konsole (bei der Installation noch am besten Desktop-Icon anwählen),
die einige Vorteile, unter anderem, der Übersichtlichkeit bietet. Ihr müsst euch z.B. nur einmal mit GitHub anmelden.
