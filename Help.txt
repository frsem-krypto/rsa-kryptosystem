1.Aktuellste Version des Verzeichnisses downloaden:
cd rsa-kryptosystem

Projekt downloaden:
git clone https://github.com/PixelProfi/rsa-kryptosystem.git

Datei pullen(aus dem Internet aktualisieren):
git pull

2. Datei öffnen und bearbeiten.

3. commiten:
git commit -m "Info"

4.Bei Fehlermeldung sicherheitshalber noch einmal pullen.

5.Datei pushen (In die Cloud wieder hochladen):
git push
-> Benutzername
-> Password



Alle Commands:
git clone https://github.com/PixelProfi/rsa-kryptosystem.git
ls = list

git pull
git push
